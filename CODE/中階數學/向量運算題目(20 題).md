# 一、基本向量運算（加減、純量乘法）

若 u = (3, −1, 2)、v = (−2, 4, 1)，求 u + v。

若 a = (5, 2)、b = (−3, 7)，求 2a − 3b。

給定向量 x = (4, −6)，求其長度 ‖x‖。

若 p = (8, −4, 2)，求單位向量 p̂ = p / ‖p‖。

# 二、內積（Dot Product）

若 u = (1, 2, −1)、v = (3, 0, 4)，求內積 u · v。

已知向量 a = (x, 2) 與 b = (3, 1) 的內積為 7，求 x。

若向量 u = (5, k) 與 v = (1, 2) 互相垂直，求 k。

# 三、外積（Cross Product）

若 u = (2, −1, 3)、v = (4, 0, −2)，求外積 u × v。

已知向量 A = (a, 1, 2) 與 B = (1, −1, 1) 的外積結果為平行於 (1, 1, 0)，求 a。

# 四、夾角、投影（Projection）

給定 u = (3, 4)、v = (−4, 3)，求兩向量的夾角。

若 a = (2, 1, −2)，求 a 在 b = (1, 0, 1) 上的投影向量 proj₍b₎(a)。

若 u = (1, 2)、v = (2, −1)，判斷它們是否互相垂直。

# 五、線性組合、線性獨立（Linear Combination / Independence）

判斷向量組 (1,2) 與 (2,4) 是否線性獨立。

對向量 v = (5,1)，找出常數 α、β，使得
  v = α(1,0) + β(1,1)。

若 u = (2,1,0)、v = (1,−1,1)、w = (3,0,1)，判斷 w 是否可由 u、v 線性組合而成。

# 六、三維向量幾何（距離、面積、體積）

求點 A(2,3,1) 與 B(−1,−1,4) 之間的距離。

已知三點 A(1,0,0)、B(2,1,0)、C(0,1,1)，求三角形 ABC 的面積。

求向量 u = (1,2,3)、v = (2,−1,0)、w = (0,1,4) 所張成平行六面體體積（混合積 |u·(v×w)|）。

# 七、平行、正交、共線判斷

若 a = (6, −3)、b = (−2, 1)，判斷是否平行。

若 p = (1,2,3)、q = (2,4,6)，判斷兩向量是否共線。
