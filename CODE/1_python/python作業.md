## 資料型態及其運算:List ` [3, 5, 7, 2]`
- 列表總和:輸入一串整數（例如` [3, 5, 7, 2]`），寫程式計算並輸出列表中所有元素的總和
- 計算偶數個數:輸入一個整數列表，請計算其中 偶數 的個數並輸出。
- 最大值與最小值:給定一個整數列表，輸出其中的 最大值 與 最小值。
  - 輸出格式範例：max = 9, min = -2
- 去除重複值:
  - 給定一個列表，例如 `[1, 2, 2, 3, 3, 3, 4]`，寫程式產生一個 不含重複值 的新列表，並維持原本出現的順序。
  - 輸出：`[1, 2, 3, 4]`
- 倒序列表:
  - 輸入一個列表，請 不要使用 `list.reverse()`，改用 `切片` 或其他方式，建立一個「倒序」的新列表並輸出。
- 找出指定元素的所有索引:
  - 給定一個列表與一個目標值 x，請找出 x 在列表中出現的 所有索引位置，以列表形式輸出。
  - 例如：列表 `[1, 3, 7, 3, 2, 3]`，目標值 3 ==> 輸出應為 `[1, 3, 5]`。
- 列表平移（旋轉）:
  - 給定一個列表與一個整數 k，請將列表 向右平移 k 格，超出範圍的元素從左邊補回
  - 輸入：原列表：[1, 2, 3, 4, 5] k = 2
  - 輸出：[4, 5, 1, 2, 3]
- list comprehension 實作練習[請 使用 `list comprehension` 完成]
  - 計算列表元素的平方:
    - 輸入一個整數列表 ==> 輸出一個新列表，內容為每個元素的平方。
    - 範例：`[1, 2, 3]`→ `[1, 4, 9]`
  - 過濾奇數:給定一個整數列表，請產生一個新列表，只保留其中的 奇數 
- 計算子列表長度:
  - 給定一個「二維列表」，例如：`[1, 2], [3, 4, 5], [6]]`
  - 請輸出每個子列表的長度，結果為 `[2, 3, 1]`
- 合併兩個列表（交錯合併）
  - 給定兩個長度相同的列表： `a = [1, 3, 5]` `b = [2, 4, 6]`
  - 輸出:新列表 `[1, 2, 3, 4, 5, 6]`

```python

    print("通過")
else:
    print("沒通過")
```
